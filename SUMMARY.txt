📋 RÉSUMÉ DE CE QUI A ÉTÉ CRÉÉ

Cette application web permet à votre amie de gérer ses histoires avec des 
intrigues parallèles, des dépendances d'événements, et un système de tags colorés.

🗄️ BASE DE DONNÉES
================================================================================

Tables créées (PostgreSQL):
- author                [Utilisateurs avec bcrypt password]
- project               [Projets/histoires par utilisateur]
- intrigue              [Intrigues parallèles par projet]
- tag                   [Tags/catégories colorés par intrigue]
- element               [Scènes/marqueurs par intrigue]
- element_tag           [Junction table pour many-to-many]
- dependency            [Dépendances entre éléments]

Indexes créés pour performance:
- idx_project_author
- idx_intrigue_project
- idx_element_intrigue
- idx_tag_intrigue
- idx_element_tag_element
- idx_element_tag_tag
- idx_dependency_element
- idx_dependency_blocked

Données de test incluées:
- 1 utilisateur: test_home / test_home
- 2 projets exemple
- 4 intrigues
- 7 éléments
- 4 tags
- 2 dépendances
